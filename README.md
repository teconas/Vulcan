# Vulcan
An opinionated C# Extension Library for Common Tools, Utilies and Extension Methods.

## Why?
Because I often reuse these.

## Whats in it?
- Extension methos for Fluency (Method-Chaining)
- Extensions Method commonly missing
- Some Helper
- Some Data Structures

## What is it for?
It is meant to be general purpose, not software, framework or case specific.
It should Work with ASP.NET, Unity and other C# Usage.
